# Study from Lecture

**ADHD-visual edition. Extract key concepts from lectures, visualize first, practice second.**

## What It Does

Converts lecture material into color-coded visual learning:
- **Concept map** (5 big ideas, color-coded, clickable)
- **Practice questions** (5 total: 2 easy, 2 medium, 1 hard)
- **Visual answers** (diagram first + 3-bullet explanation + memory hook)
- **Color-coded cheat sheet** (scannable one-pager, formulas, visual icons)
- **Test mode** (5-question visual quiz with reveal cards, progress bar, scoring)

## Key Features

- **Pictures before text** — diagrams render first, prose second
- **Color consistency** — same concept = same color throughout (teal for definitions, purple for formulas, amber for traps)
- **3 bullets max** — every answer under 3 short bullets, one bold key word per bullet
- **Memory hooks** — vivid one-liner to remember each concept
- **Clickable diagrams** — tap colored boxes to pull deeper info
- **Test scoring** — visual progress bar, got-it / review-again buttons, summary at end
- **No long paragraphs** — all chunked and pattern-based

## When to Use

- *"Help me study this PDF"*
- *"What are the key concepts from this lecture?"*
- *"Make me a visual study guide"*
- *"Quiz me on these slides"*

## Input

- PDF (lecture notes, textbook chapters)
- PowerPoint slides
- Handwritten notes
- HTML lecture material

## Output

Concept map → practice questions → visual answers → cheat sheet → optional test mode.

## What's Special

**Hyperfocus bait** — most interesting concept is placed first, not in lecture order.

**Tangent harvesting** — if you ask an adjacent question mid-quiz, Claude answers it and offers it as a bonus question instead of redirecting you.

**Test mode clarity** — each question gets its own card with difficulty color stripe, big bold question, tap-to-reveal answer format.

**Scannable cheat sheet** — only the concepts you need, max 6 boxes, icons + formulas.

## Handoff

After the cheat sheet or test wrap-up, if math-heavy, offers: *"Want to drill this?"* → Hands off to **Math Fluency Drill**.
