# Learning Recall

**Lock in long-term memory through forced retrieval (active recall) — not re-reading.**

## What It Does

Forces you to **pull answers from a blank page** before checking them:
- **Blank-page recall targets** — one simple concrete fact per question
- **You answer first** — no hints, no multiple choice
- **Wrong answer → re-teach** (study-from-lecture style: picture + 3 bullets + memory hook)
- **Re-queue missed chunks** — comes back after 3–5 other questions (spaced retrieval)
- **Session wrap-up** — shows what you got first try vs. took two tries vs. still shaky
- **Spacing schedule** — tells you: revisit in 1 day, 3 days, 1 week (that's what locks it in)

## Why This Works

**Retrieval is the learning event**, not re-reading.

The hippocampus captures information fast, but it's fragile. Consolidation — transferring it into durable cortical storage — is driven by *retrieval*, not by re-exposure. Every time you pull an answer from a blank page, you strengthen the trace. Reviewing notes again doesn't do this; rebuilding the answer from scratch does.

## When to Use

- *"Test my recall on this paper"*
- *"Drill me on these slides"*
- *"I need to memorize this material"*
- *"Active recall practice"*
- **After** you've already seen the material (study-from-lecture, STEM Learning App, etc.)

## Input

- Research paper (already read)
- PowerPoint slides (already studied)
- PDF lecture notes (already reviewed)
- Any material you've *already seen once*

## Output

Recall session → re-teach on miss → spacing schedule → next session timing.

## What's Special

**No answer hints** — you have to pull it from memory. That's the work.

**Re-teaching on miss** — wrong answer triggers a study-from-lecture style mini-lesson (visual first, 3 bullets, memory hook), then re-queues it.

**Within-session spacing** — missed problems come back 3–5 questions later, reworded slightly. That's real spaced retrieval, not cramming.

**Tangent harvesting** — ask an off-topic related question? Answer it, turn it into a bonus recall target. Don't suppress curiosity.

**Plain text, fast loops** — speed and momentum matter. Minimal decoration, max momentum.

## Relationship to Other Skills

- **study-from-lecture** = first contact. Teaches new material with visuals + practice.
- **learning-recall** = retention. Forces retrieval of material you've already seen; on a miss, uses study-from-lecture's teaching pattern.

If you haven't seen the material at all, study-from-lecture first. Then come back here to lock it in.
