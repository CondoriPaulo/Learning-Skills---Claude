# ML Math Tutor v2

**Patient, structured math tutor that anchors ML concepts directly to your code and dataset.**

## What It Does

Breaks down machine learning math into 8 interconnected teaching blocks:
1. **Linear Algebra Foundations** (vectors, matrices, norms)
2. **Calculus Foundations** (derivatives, gradients, chain rule)
3. **Geometry of Model Fitting** (residuals, constraints, regularization)
4. **Linear Regression & Metrics** (OLS, RMSE, R²)
5. **Ridge, Lasso, ElasticNet** (regularization + feature scaling)
6. **Gradient Descent & SGD** (optimization + soft thresholding)
7. **Model Evaluation** (train/val/test, hyperparameter tuning, bias-variance)
8. **Synthesis** (connect theory to your actual results)

## Key Features

- **WHAT / HOW / WHY method** — definition → concrete numbers → why it matters in *your code*
- **Symbol Ledger** — every notation explained once, tracked throughout
- **Hyperfocus bait** — you pick which block to start with (don't force driest first)
- **Two quiz modes**: LEARN (2 questions, deep) or SHIP (1 micro-check, fast)
- **Rescue path** — miss twice → re-explain simpler, reveal answer, restate back
- **Tangent capture** — go off-topic mid-session? Answer it, fold it back in
- **Recall ladder** — start with 3 recall questions from prior blocks

## When to Use

- *"Help me understand the math in my regression assignment"*
- *"I have this loss function — walk me through the gradient"*
- *"Why does ElasticNet do this but Ridge doesn't?"*
- *"My R² is negative — what does that mean?"*

## Input

- Assignment text or question
- Your code (Python, R, notebooks)
- Dataset description or sample stats
- Model results (RMSE, coefficients, etc.)

## Output

Structured block-by-block teaching → quiz → optional fluency drill → synthesis.

## What's Special

**Anchored to YOUR code**, not abstract. You say β = my coefficient — Claude uses that, not generic notation.

**Chunked for ADHD** — max 5–8 min reading per response. One concept at a time.

**Recall spiral** — spaced retrieval built in: start session with 3 recall questions from 1–2 blocks back.

## Handoff

After math-heavy quiz blocks, offers: *"Want to drill this?"* → Hands off to **Math Fluency Drill**.
