# STEM Learning App

**Transform any STEM lecture, slide deck, or textbook into an interactive, self-contained HTML learning course.**

## What It Does

Takes a PDF, slides, or notes and converts it into a **single .html file** — an ADHD-friendly, gamified interactive course with:
- **Teach-first flow** per concept: The Idea → See It → Worked Example → Now You Try
- **Bionic reading** (bold-first 40% of letters for faster scanning)
- **XP, levels, streaks** — rewards every 30–60 seconds
- **Worked examples** — step-by-step, color-coded, revealed one line at a time
- **Quiz questions with hints** — spaced after teaching, never before
- **Rabbit holes** — optional "Go deeper?" bonus cards that let curiosity happen productively
- **Mobile-first** — works on phones, offline, no dependencies

## When to Use

- *"Turn this lecture into an interactive app"*
- *"Make me a STEM learning game from this PDF"*
- *"Convert these slides into a course I can learn from"*
- *"Make this ADHD-friendly so I can actually focus"*

## Input

- PDF (lecture, textbook chapter, course notes)
- PowerPoint / slides
- HTML lecture notes
- Any STEM topic (math, ML, physics, stats, CS, engineering)

## Output

One `.html` file. Open it. Learn. Works offline.

## What's Special

**Hyperfocus bait:** The most interesting concept is placed *first*, not in document order — gets the dopamine flowing before drier material.

**Rabbit holes harvested:** Going off-track is encouraged via bonus cards, not punished.

**No blank screens:** Static reading mode always visible if JavaScript fails (hardened for iOS Quick Look preview).

**Math fallback:** Equations render as Unicode if MathJax doesn't load — always readable.

## Handoff

After "Lock it in" on math concepts, offers: *"Want to drill this with practice problems?"* → Hands off to **Math Fluency Drill**.
